# Sistema de Vendas em Python

Sistema de vendas desenvolvido em Python para execução no terminal, com foco em treino de lógica de programação, organização de código e interação com o usuário.

# Funcionalidades
- Sistema de login com limite de tentativas  
- Barra de progresso simulando carregamento  
- Catálogo de produtos  
- Carrinho de compras  
- Cálculo automático do total  
- Desconto automático para compras acima de R$100  

# Objetivo do Projeto
Projeto criado para fins de aprendizado e prática em Python, lógica de programação e desenvolvimento de sistemas simples.

# Tecnologias Utilizadas
- Python 3
